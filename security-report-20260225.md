### 1. 代理信任配置缺失
**问题**：`gateway.trustedProxies` 为空
**修复**：如需反向代理，配置信任的代理 IP

### 2. 防火墙规则
**问题**：UFW 已启用但规则需确认
**当前状态**：
- SSH 端口 22 对所有接口开放
- 端口 5555 对所有接口开放（需确认用途）
**修复建议**：
```bash
# 限制 SSH 仅本地网络
sudo ufw allow from 192.168.0.0/16 to any port 22

# 如不需要端口 5555，关闭它
sudo ufw deny 5555
```

### 3. SSH 根登录
**问题**：`PermitRootLogin yes`
**修复建议**：
```bash
# 编辑 SSH 配置
sudo sed -i 's/^PermitRootLogin yes/PermitRootLogin no/' /etc/ssh/sshd_config
sudo systemctl restart sshd
```

### 4. 扩展插件未限制
**问题**：`plugins.allow` 未设置
**修复**：在配置中明确列出信任的插件 ID

### 5. 插件版本未固定
**问题**：飞书插件版本未固定
**修复**：固定插件版本到具体版本号
